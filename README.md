# ClassHub - School Management App

A production-ready Flutter application for managing students, classes, teachers, attendance, and announcements.

## Features

- Student directory with local database
- Manual attendance marking (check-in/check-out)
- Teacher announcements and schedule calendar
- Parent messages (local storage)
- Photo attachments via image picker
- Export reports via Storage Access Framework (SAF)
- Role-based UI (Teacher/Admin)
- Accessibility-first design
- Offline-first architecture
